## Configuration

### Example

**`_config.yml`**:

```yaml
lsi: false
safe: true
source: [your repo's top level directory]
incremental: false
highlighter: rouge
gist:
  noscript: false
kramdown:
  math_engine: mathjax
  syntax_highlighter: rouge
```

  * You can append `?ts=2` or `?ts=4` to the URL to change the tab-size.

  * Add stylish extension

  ```css
  * { tab-size: 4 }
  ```
