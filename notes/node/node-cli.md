## Install Locally

```bash

```

## Troubleshooting

[Running global npm scripts in node 12.9.0: This command not found error. #29287](https://github.com/nodejs/node/issues/29287#issuecomment-524859390)

Delete the following folders:

```none
%AppData%\npm
%AppData%\npm-cache
```
