# Pug (Previously Jade)

https://pugjs.org/

> Pug is a high performance template engine heavily influenced by Haml and implemented with JavaScript for Node.js and browsers.



## [Iteration](https://pugjs.org/language/iteration.html)

```pug
ul
  each val in [1, 2, 3, 4, 5]
    li= val
```

```html
<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
  <li>4</li>
  <li>5</li>
</ul>
```
