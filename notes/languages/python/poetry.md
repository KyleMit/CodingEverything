# Poetry

<https://python-poetry.org/>

> Python Packaging and Dependency Management Made Easy

## Docs

* Installing Tool

  ```bash
  pip install poetry
  ```

* Installing Packages

  ```bash
  poetry install
  ```

* [Activating the virtual environment](https://python-poetry.org/docs/basic-usage/#activating-the-virtual-environment)

  ```bash
  poetry shell
  ```
