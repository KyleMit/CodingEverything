## Doc Comments

Emmet autocomplete `/**`

## Params

```js
/**
 * @param {<type>} <paramName>
 * /
```

```js
/**
 * @param {number} a
 * @param {number} b
 */
function add(a, b) {
  return a + b
}
```

## Usage With TypeScript

[TypeScript - JSDoc Reference](https://www.typescriptlang.org/docs/handbook/jsdoc-supported-types.html)

```ts

 * @type {string}
 */
var s;
```
