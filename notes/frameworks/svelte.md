## Questions

* [Can we write typescript inside a svelte component?](https://stackoverflow.com/q/57034573/1366033)

  Add `lang="ts"` to your `script` block

  ```svelte
  <script lang="ts">
    export const hello: string = 'world';
  </script>
  ```
